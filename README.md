🌐 Visualization of 3D Point Clouds
📌 About

This application is designed to read and visualize 3D point cloud data from XYZ-formatted files. It provides a simple and intuitive way to explore spatial datasets in a three-dimensional environment.

Currently, the application supports .txt input files containing XYZ coordinates. Support for additional file formats is planned for future releases.

🚀 Features
📥 Import point cloud data from XYZ-formatted .txt files
🧭 Interactive 3D visualization of point clouds
⚡ Lightweight and easy-to-use interface
🔜 Planned support for additional formats (e.g., .csv, .xyz, .las)

🌍 Application Domain
General-purpose 3D data visualization
Forestry and environmental analysis
Photogrammetry and LiDAR data exploration
